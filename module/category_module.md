# Category Module

## Purpose
จัดการหมวดหมู่สินค้า

## Responsibility
- เพิ่มหมวดหมู่
- แก้ไขหมวดหมู่
- ลบหมวดหมู่
- จัดลำดับหมวดหมู่
- เปิด/ปิดหมวดหมู่

## Input
- Category API
- Category Table

## Output
- Category List
- Category Tree

## Event
- CATEGORY_CREATE
- CATEGORY_UPDATE
- CATEGORY_DELETE
- CATEGORY_STATUS_CHANGE

## Relation
- Product Module
- Search Module

## Future Expansion
- Multi Level Category
- Category Image
- Category SEO
