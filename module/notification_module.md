# Notification Module

## Purpose
จัดการการแจ้งเตือนของระบบ

## Responsibility
- แจ้งเตือนคำสั่งซื้อ
- แจ้งเตือนการชำระเงิน
- แจ้งเตือนการจัดส่ง
- แจ้งเตือนสมาชิก

## Event
- ORDER_CREATED
- PAYMENT_SUCCESS
- SHIPPING_SENT
- MEMBER_REGISTERED

## Future Expansion
- Email Notification
- SMS Notification
- LINE Notification
- Push Notification
