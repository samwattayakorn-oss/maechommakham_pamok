# Mae Chor Makham (แม่ช่อมะขาม)

## Project

เว็บไซต์สำหรับนำเสนอสินค้า รับออร์เดอร์ และบริหารการขายของร้านแม่ช่อมะขาม

เป้าหมายของโครงการคือการสร้างระบบที่เรียบง่าย มั่นคง ขยายต่อได้ และดูแลรักษาได้ง่าย โดยใช้โครงสร้างแบบโมดูล (Module) เพื่อให้สามารถพัฒนาทีละส่วนได้อย่างต่อเนื่อง

---

## Objective

- แนะนำสินค้า
- รองรับการสั่งผลิตตามออร์เดอร์
- เพิ่มความน่าเชื่อถือของร้าน
- ลดขั้นตอนการดูแลระบบ
- รองรับการขยายในอนาคต

---

## Design Principle

โครงการนี้พัฒนาภายใต้หลักการดังต่อไปนี้

- โครงสร้างมาก่อนรายละเอียด
- หนึ่งโมดูล หนึ่งหน้าที่
- ใช้ข้อมูลร่วมกันเท่าที่จำเป็น
- ลดการทำงานซ้ำ
- เพิ่มความสามารถในการขยาย
- ปรับปรุงได้โดยไม่กระทบทั้งระบบ

## Module

โมดูลแต่ละตัวมีหน้าที่เฉพาะของตนเอง และสามารถพัฒนา แก้ไข หรือเพิ่มเติมได้โดยไม่กระทบกับโมดูลอื่น หากเป็นไปได้

ตัวอย่าง

- Product
- Review
- Order
- Shipping
- Customer
- Admin

---

## Shared

ข้อมูลที่ใช้ร่วมกันจะถูกแยกออกจากโมดูล เพื่อให้ทุกส่วนสามารถอ้างอิงข้อมูลเดียวกัน ลดความซ้ำซ้อน และลดภาระในการดูแลรักษา

## Philosophy

เว็บไซต์นี้ไม่ได้มุ่งสร้างระบบที่ซับซ้อนที่สุด

แต่มุ่งสร้างระบบที่

- ใช้งานได้จริง
- ดูแลได้จริง
- ขยายได้จริง
- พัฒนาได้ต่อเนื่อง

โดยให้โครงสร้างเติบโตไปพร้อมกับการเติบโตของธุรกิจ
