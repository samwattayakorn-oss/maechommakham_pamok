# แม่ช่อมะขาม

## ความหมาย
เว็บไซต์สำหรับนำเสนอสินค้า บริการ และการสื่อสารกับลูกค้า
โดยออกแบบบนแนวคิด SAFE
(Sam Atom Framework Evolution)

## เป้าหมาย
- โครงสร้างเรียบง่าย
- ขยายได้
- โมดูลแยกอิสระ
- ใช้ข้อมูลร่วมกัน
- ลดความซ้ำซ้อน
- พัฒนาต่อได้ไม่สิ้นสุด

## หลักการ

↓

↓

↓

↓
